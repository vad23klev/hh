Скрипт рассылки WR-Subscribe 1.2
:: Назначение скрипта ::
WR-Subscribe - скрипт позволяющий создать на Вашем сайте рассылку любой информации: (Новости, анекдоты, ссылки на программы, объявления сайта и т.д.). Для получения рассылок подписчик должен ввести свой емайл в поле подписки. Рассылка производится администратором сайта из админпанели. Формат рассылаемых сообщений - html.
Скрипт компактен, прост в использовании, максимально универсален в настройке. Работает при любых значения register_globals и error_reporting Вашего хостера. В качестве базы данных использует файлы. Что позволяет легко перенести скрипт на другой сайт/хостинг/папку. Размер скрипта: - 30,0 Кб.
:: Возможности ::
Универсальная панель администратора с возможностями:
    - просмотра/удаления/добавления подписчиков;
    - сохранение для последующего просмотра (как в html так и в виде кода) отправленной рассылки;
    - наличие модуля отписки от рассылки;
Изменения в версии 1.1:
- сохранена БД без изменений. Скрипт v. 1.1 на 100% совместим с предыдущей версией;
- Изменён механизм отправки рассылки. Теперь отправка производится этапами. Шаг задаётся в админпанели. Рекомендуемый размер шага - 20. Скрипт теперь корректно работает при любом кол-ве подписчиков - хоть миллион ;-)
- Закрыта XSS-уязвимость.
:: Установка и использование ::
1. Создайте папку на Вашем сервере, например "subscribe". Закачайте в эту папку все файлы.
Выставите ей атрибуты Chmod 755
Закачивать и настраивать атрибуты скрипта можно менеджером "Total Commander".(Файл -> Изменить атрибуты).
2. Далее указан состав скрипта и права доступа, которые необходимо выставить файлам:

Файлы / описаниеПрава доступа
  • index.html - страница с формой подписки;
    		• 755
  • addemail.php - страница подписки/отписки от рассылки;
  • admin.php - cтраница администратора. Пароль - admin;
  • README.html - этот файл;
    		• -
  • config.php - файл с конфигурационными данными;
    		• 644
  • database.php - файл с данными подписчиков;
  • allsubscribe.php - файл с отправленными рассылками;
  • msg.dat - временный файл. Используется при рассылке.

    • Увидеть форму для подписки можно по ссылке: http://Мой_сайт/subscribe/index.html
    Но, лучше всего запустить админпанель : http://Мой_сайт/forum/admin.php. Пароль по умолчанию admin и из неё получить универсальную форму для подписки (в ней будет указан абсолютный путь к скрипту). Полученная форма будет работать на любой странице Вашего сайта.
    3. На локалхосте права доступа ставить не нужно. Права необходимо выставлять только на серверах в Интернете. Данные права доступа используются на большинстве серверов. В случае, если скрипт не запускается и выводится ошибка "Permise Denied" - скорее всего Вы неверно установили права или что то сделали не так.
    Можете посмотреть статью по настройке прав доступа.

    Если скрипт не заработал, не бойтесь экспериментировать: изменяйте атрибуты каталога и файлов.
    Иногда, на "сомнительном" хостинге, скрипт запускается с атрибутами 777 (вместо 755) и 666 (вместо 644) для файлов и папки "subscribe" соответственно согласно таблице. Либо 777 для всех файлов.
    P.S. Внимание! Скрипт написан на PHP и не работает на бесплатном хостинге НЕ поддерживающего эту язык, например, www.narod.ru.

    Если все пункты вы выполнили, а скрипт так и не запустился, ну чтож, заходите на форум по адресу: http://www.wr-script.ru/forum/ и в разделе "WR-Scripts" просмотрите разделы форума, возможно решение Вашей проблемы уже есть на форуме. Если нет - задавайте вопросы. Постарюсь Вам помочь.

    Внимание! Не увлекайтесь рассылками писем. При определённых условиях массовую рассылку писем можно трактовать как рассылку СПАМа! А это уже административно (а в некоторых странах уголовно) наказуемое дияние. Делайте выводы.
